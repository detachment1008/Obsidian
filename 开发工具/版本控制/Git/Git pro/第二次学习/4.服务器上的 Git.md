# 4.9 第三方托管的选择
---

因为对搭建自己的 Git 仓库没有兴趣，可以直接将自己的 Git 项目托管到一个外部的专业托管网站

好处：
1. 托管网站可以快速建立并开始项目
2. 无需服务器维护和监控工作
3. 有助于开源社区发现和帮助你
