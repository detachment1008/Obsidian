1. 下载 `graphviz`
```
sudo apt install graphviz
```

2. 下载 `doxygen`
```
sudo apt install doxygen
```

3. 进入项目目录

4. 生成配置文件
```
doxygen -g Doxygen.config
```

5. 修改配置文件
```
EXTRACT_ALL            = YES
HAVE_DOT               = YES
UML_LOOK               = YES
RECURSIVE              = YES 
```

6. 生成
```
doxygen Doxygen.config
```

7. 打开 `html/index.html`
