# 哪类程序可被调试

对于 C 程序，编译时需要加 -g 参数，以保留调试信息

在调试时，表明可以调试
```
Reading symbols from a.out...(no debugging symbols found)...done.// 如果没有调试信息会提示 no debugging symbols fount
```

表明没有调试信息，无法调试：
```
Reading symbols from a.out...done.
```

# 查看段信息(readelf)

```
readelf -S a.out | grep debug
```

# 查看 strip 状况(file)

```
file a.out
```

# 调试方式运行程序(gdb)

```
gdb a.out
```

运行：
```
run 参数

set args 参数
run
```
