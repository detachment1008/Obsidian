# 7.1 定义抽象数据类型
---

类的基本思想：数据抽象
- 依赖接口和实现分离的技术

类的基本思想：封装
- 实现了类的接口和实现分离

构造函数没有返回类型，不能被声明为 const

合成默认构造函数：
- 如果存在类内的初始值，用它来初始化成员
- 否则，默认初始化该成员
- Sales_data() = default;

可以用初始化列表进行初始化

# 7.2 访问控制与封装
---

友元函数只能出现在类内部，且具体位置不限
- 但它不是类的成员，也不受它所在区域访问控制级别的约束
