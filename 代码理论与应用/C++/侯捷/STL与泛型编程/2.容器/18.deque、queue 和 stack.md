# deque

它实际是分段连续的，即段内连续，段间不连续。但是使用者还是认为它是完全连续的。

它使用了一个 vector 用以存储指向每个段的指针，每个段相同大小，排列由 vector 存储的指针控制。

它的迭代器存储 4 个元素：
1. node：指向当前段的指针
2. first：当前段的段首
3. last：当前段的段尾
4. cur：当前所在元素的位置

它的第三个泛型参数就是指定的缓冲区大小

它的 insert 函数很聪明，它会判断是前面的元素少还是后面的元素少，然后移动元素少的那一个部分
