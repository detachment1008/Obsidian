# 转换函数
---

即控制类型转换的函数

```
class Fraction
{
public:
	opeartor double() const
	{
		return 3.1415926;
	}
}
```

即用 operator 和 double 作为函数名：
- 即当改对象被当成double用的时候，就调用这个函数来获取它的double类型
	- 这个 double 不一定是基础类型
	- 它可以是任何类型
- 不能有参数
- 可以不写返回值，因为在函数名中就已指出
- 只有转换函数中，函数名中间可以有空格
- 这种函数通常都会使用const函数

步骤：
- 找是否有运算符重载
- 找是否有转换函数

# 小总结
---

目前对于 operator 我们已经知道了两种用法：
- 运算符重载
	- operator 需要和运算符紧紧相连作为函数名
- 转换函数
	- operator 需要和类型名中间隔开一个空格
	- 只有转换函数的函数名中可以有空格
