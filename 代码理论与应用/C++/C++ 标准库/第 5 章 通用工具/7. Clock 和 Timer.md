# Chrono

它是一个日期和时间的库，定义在 `chrono`
命名空间：`std::chrono`

## Duration 时间段

它表现为一个数值和一个分数的组合
```
std::chrono::duration<int> a(20);                        // 秒为单位
std::chrono::duration<double, std::ratio<60>> b(0.5);    // 分为单位
std::chrono::duration<long, std::ratio<1, 1000>> c(1);   // 毫秒为单位
```

使用：
```
dura.count() // 获取所有的 tick 数量
```

## Clock

表示一个起点和一个 tick 周期

## Timepoint
