`using` 在模板的地方比 `typedef` 强，因为别名也可以模板化

如：使用了自定义分配器的模板别名
```cpp
template <typename T>
using MyAllocList = std::list<T, MyAlloc<T>>;
```

使用：
```cpp
MyAllocList<Widget> lw;
```

`typedef`：
```cpp
template <typename T>
struct MyAllocList
{
	typedef std::list<T, MyAlloc<T>> type;
}

MyAllocList<Widget>::type lw;
```
