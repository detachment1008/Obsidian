# `leaq`

通过内存的方式计算数值，但不获取内存的值

它可以计算形如：`x + y + s * t` 这样形式的式子

# 一元操作

即它的操作数既是源，又是目的

## `inc`

自增 1:  inc D
```
incb
incw
incl
incq
```

## `dec`

自减 1: dec D
```
decb
decw
decl
decq
```

## `neg`

取负: neg D
```
negb
negw
negl
negq
```

## `not`

取补: not D
```
notb
notw
notl
notq
```
