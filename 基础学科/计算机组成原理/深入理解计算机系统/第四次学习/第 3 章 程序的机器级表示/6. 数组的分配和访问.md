# 一维数组

`(起始地址, 索引, 类型大小)`：这种内存访问格式，非常适合数组类型的元素访问

指针的加减运算，会自动根据其类型的大小进行伸缩，即 `int *t; t + 1 == t + 1 * sizeof(int)`

# 二维数组

嵌套数组：
```
T D[R][C];
&D[i][j] = Xd + L(C*i + j)
```

如果要应用左边的索引，需要额外乘以右边的维度，如果要应用右边的索引，加上即可； 然后根据指针的伸缩，再乘以类型的字节大小即可
