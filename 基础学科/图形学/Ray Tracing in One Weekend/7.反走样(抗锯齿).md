# 生成随机数

利用 cstdlib 中的 rand 函数
- 它是伪随机数
- 它在 0 到 RAND_MAX 的范围内生成一个随机整数

伪随机数：
- 看起来是随机数，但是它是通过一个确定的算法生成的数字序列，所以给定种子的情况下，它生成的随机序列是相同的
- 每次可以重新设置随机数的种子，从而尽量的让他更加随机
	- `time(NULL)`：当前系统的时间(从1970年1月1日到现在的秒数)
	- `srand`：设置种子
	- 默认种子是 1
